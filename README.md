这是一个参（抄）考（袭）了bilibili和隔壁各个bgm客户端的又双叒叕一个[Bangumi番组计划](https://bgm.tv)Android客户端

## 现在有啥功能

更完整的收藏列表

更详细的放送时间表

更好用的番组搜索/索引

更全面的番组详情

更快捷的收藏/进度管理

原生体验的超展开/时间胶囊

## 最近更新了啥

\[实验性]加载时渲染，更快的加载速度

\[实验性]帖子回复支持富文本编辑

半透明导航栏
